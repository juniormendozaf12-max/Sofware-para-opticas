FORMAS DE ABRIR EL SISTEMA:
============================

OPCION 1 - FACIL (Recomendada)
-------------------------------
Doble clic en: INICIAR_SISTEMA.bat
- Te permitira elegir el navegador
- Funciona con Chrome, Edge, Brave o navegador por defecto

OPCION 2 - ABRIR EN NAVEGADOR ESPECIFICO
-----------------------------------------
Doble clic en el archivo correspondiente:
- Abrir_en_Chrome.bat   (Para Google Chrome)
- Abrir_en_Edge.bat     (Para Microsoft Edge)
- Abrir_en_Brave.bat    (Para Brave Browser)

OPCION 3 - SERVIDOR LOCAL (Mejor rendimiento)
----------------------------------------------
Doble clic en: Iniciar_Servidor_Local.bat
- Inicia un servidor HTTP local
- Abre tu navegador y ve a: http://localhost:8000/Centro_Optico_Sicuani_v5_Purple.html
- MANTENER LA VENTANA ABIERTA mientras uses el sistema
- Requiere Python instalado (descarga en https://www.python.org)

OPCION 4 - DOBLE CLIC DIRECTO
------------------------------
Doble clic en: Centro_Optico_Sicuani_v5_Purple.html
- Se abrira en tu navegador por defecto
- Puede que algunos navegadores bloqueen localStorage

SOLUCION DE PROBLEMAS:
======================

Problema: "No puedo ver las pestanas de Medidas, LC, etc."
Solucion: El modal ahora es mas grande (1050x700px). Asegurate de tener resolucion minima de 1366x768

Problema: "La busqueda no funciona"
Solucion: Escribe al menos 2 caracteres. Prueba con "Aldo" o "12345678"

Problema: "No guarda los datos"
Solucion:
- Usa el servidor local (Iniciar_Servidor_Local.bat)
- O abre desde Chrome/Edge con permisos de localStorage

Problema: "Aparece pantalla en blanco"
Solucion: Presiona F5 para recargar. El sistema hace auto-login automaticamente.


SOPORTE:
========
Si tienes problemas, verifica:
1. Resolucion de pantalla minima: 1366x768
2. Navegador actualizado
3. JavaScript habilitado
4. Presiona F12 y revisa la consola para ver mensajes de debug
